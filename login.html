GitHub Classroom Helper — UTN FRC

Autenticación vía GitHub OAuth. Si querés usar una cuenta distinta, ingresá el usuario de GitHub para forzar el cambio.

Abre GitHub para autorizarte. Al cerrar sesión podés cambiar de cuenta ingresando un usuario distinto arriba.

Ingresá un Personal Access Token de GitHub. Es la forma más rápida para cambiar de cuenta o entorno sin depender del navegador.

¿Cómo generar un token?
  1. GitHub → Settings → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Generate new token
  4. Permisos: read:org, repo, read:user
